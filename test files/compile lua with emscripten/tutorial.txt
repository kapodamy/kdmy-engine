Steps:
1. download the Lua sourcecode
2. edit the "src/Makefile":
	replace "LUA_T= lua" with "LUA_T= lua.a"
	replace the block of "$(LUA_T): $(LUA_O) $(LUA_A)" with:

$(CC) -o $@ $(LDFLAGS) $(LUA_O) $(LUA_A) $(LIBS) -s "ALLOW_MEMORY_GROWTH=1"

3. copy "_build.sh" file to "src" folder
4. open "src" in msys2 console and run "./_build.sh"
5. grab only the "liblua.a" file then copy into "luascript-js/lib" folder


notes:
* msys2 and emscripten must be installed first
* example of modified Makefile of lua 4.6 added for reference
* updating emscript can make obsolete "liblua.a", to solve recompile again
